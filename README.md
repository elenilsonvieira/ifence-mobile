# IFence-mobile
O IFence é um aplicativo de monitoramento de crianças em ambientes abertos.

# Tecnologias Utilizadas
- React Native
- Expo
- AsyncStorage 
- React Native Maps

# Pré-Requisitos
- Node.js
- Expo CLI (se estiver usando Expo)
- Git
- Um emulador Android/iOS ou dispositivo físico(Opcional, o aplicativo Expo Go disponibliza um um emulador)
